# FAQ Chatbot

An AI-powered FAQ Chatbot that answers questions using cosine similarity.

## Features
- NLP-based intent matching using cosine similarity
- Typing indicator animation
- Suggestion chips for quick questions
- Confidence score for each answer
- Clean chat UI

## Tech Stack
- HTML, CSS, JavaScript

## How to Run
1. Download the repository
2. Open `index.html` in any browser
